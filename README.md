# Instructions

## Python Solutions

### Setup

- Install `pyenv` and `pipenv`.
- Run `pipenv install --ignore-pipfile`.
- Run `pipenv shell`.

### Verifying Solutions

From within the day's folder, run `pipenv run python <path/to/source>.py`.

## Kotlin Solution

### Verifying Solutions

From within the `22Kotlin` folder, run `./gradlew run`.
